Ejercicios de Estadística

Ejercicio 2 - Prueba t de una muestra

Un laboratorio farmacéutico desarrolla una nueva pastilla para reducir el tiempo de reacción.

  • μ₀ = 850 ms
  • n = 18
  • x̄ = 812 ms
  • s = 74 ms
  • α = 0.05
t = (x̄ - μ) / (s / √n)

Ejercicio 3 - Intervalo de confianza

Una fintech desea estimar la proporción de clientes que pagan correctamente sus microcréditos.

  • n = 400 clientes
  • 312 pagaron correctamente
  • Confianza = 95%
IC = p ± Z * √((p(1-p))/n)

Distribución Normal Z

La distribución normal se utiliza para pruebas de hipótesis e intervalos de confianza cuando se conoce la desviación estándar poblacional.

Distribución t-Student

La distribución t-Student se utiliza cuando la muestra es pequeña y no se conoce la desviación estándar poblacional.
